# 🎨 Turtle Graphics Drawing

A simple Python project using the built-in `turtle` module to create a colorful geometric drawing.

---

## 🖌️ Overview

This program uses Python Turtle Graphics to draw a shape combining:

* A red triangle 🔺
* Black lines for structure
* Green and beige colored sections for design

It demonstrates basic drawing concepts like movement, rotation, colors, and pen control.

---

## ✨ Features

* Draws shapes using `turtle`
* Uses multiple colors 🎨
* Demonstrates:

  * `forward()` / `back()`
  * `left()` / `right()`
  * `penup()` / `pendown()`
* Simple loop for drawing repeated shapes

### 1. Clone the repository

```bash

```

### 2. Navigate to the project folder

```bash
cd turtle-drawing
```

### 3. Run the program

```bash
python turtle_drawing.py
```

---

## 🧠 How It Works

* The turtle starts facing upward.
* A triangle is drawn using a loop.
* Additional lines and shapes are drawn step by step using:

  * Direction changes
  * Position adjustments
  * Color changes

---

## 🎯 Learning Concepts

This project helps beginners understand:

* Basic graphics in Python
* Coordinate movement
* Loops and repetition
* Code structure for drawing

## ⚠️ Notes

* The `turtle` module comes pre-installed with standard Python.
